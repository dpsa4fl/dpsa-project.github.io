Expand description

Extension trait for field elements that can be converted back and forth to an integer type.

The Integer associated type is an integer (primitive or otherwise) that supports various arithmetic operations. The order of the field is guaranteed to fit inside the range of the integer type. This trait also defines methods on field elements, pow and modulus, that make use of the associated integer type.
